# Блок 1 — Быстрые фиксы

> Точечные баги и мелкие визуальные правки. Одна сессия.

## 1. Аналитика кабинета: не все модели в графике затрат

**Где:** Личный кабинет → Аналитика
**Проблема:** В графике расходов показываются затраты не на все модели. Нужно разобраться детально — какие модели пропускаются и почему.
**Файлы для старта:**
- `backend/apps/cabinet/` — API аналитики
- `frontend/app/(workspace)/cabinet/` — страница кабинета

## 2. Прогресс-бар хранилища — тусклый цвет на светлой теме

**Где:** Личный кабинет → Хранилище
**Проблема:** Цвет полоски прогресса хранилища тусклый на светлой теме. Проверить и остальные цвета кабинета на обеих темах — могут быть аналогичные проблемы.

## 3. Элементы из корня проекта не видны в промпт-баре

**Где:** Workspace → PromptBar → выбор image input
**Проблема:** При добавлении изображения через промпт-бар отображаются только элементы из групп, но не те, что лежат в корне проекта (без группы).
**Файлы для старта:**
- `frontend/components/generation/` — PromptBar, image inputs
- `backend/apps/elements/` — API элементов, фильтрация

## 4. Реакции в шеринге не ставятся в лайтбоксе

**Где:** Share page → Lightbox mode → Reactions
**Проблема:** В режиме лайтбокса на странице шеринга реакции не работают.
**Файлы для старта:**
- `frontend/app/share/[token]/page.tsx`
- `frontend/components/lightbox/`
- `backend/apps/sharing/`

## 5. Карточки проектов — увеличить в 1.5x + превью

**Где:** Список проектов, карточки групп
**Проблема:** Карточки проектов визуально маловаты. Увеличить в 1.5 раза, остальные элементы (текст, иконки, метаданные) расположить соответственно. Превью на карточках проектов и групп должны быть премиальнее — нравится тема с подложкой, на которой написаны характеристики.
**Файлы для старта:**
- `frontend/components/project/ProjectCard.tsx`, `ProjectGrid.tsx`
- `frontend/components/scene/SceneCard.tsx`
