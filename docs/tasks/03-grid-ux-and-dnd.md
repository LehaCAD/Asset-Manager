# Блок 3 — Grid UX: папки и drag & drop

> Структурные изменения в workspace: разделение папок/элементов и перемещение через DnD.

## Задачи

### 1. Разделение папок и элементов (референс: Frame)

Папки (группы) выделить в отдельную категорию наверху. Элементы — уровнем ниже. Папки живут отдельно и не зависят от настроек вкладки "Вид".

**Референс:** Frame.io — папки отделены, ассеты отдельно.

### 2. Drag & Drop элементов в группы

- Перетаскивание элементов в группы через DnD.
- Перетаскивание групп в другие группы (с учётом лимита вложенности).
- Нужен бэкенд API для перемещения элементов между группами.

**Файлы:**
- `frontend/lib/store/scene-workspace.ts` — стор workspace
- `frontend/components/element/` — DnD-Kit интеграция
- `backend/apps/elements/` — API перемещения
- `backend/apps/scenes/` — API перемещения групп

## Подход

1. Изучить текущую реализацию grid и DnD
2. Спроектировать API перемещения (backend)
3. Нарисовать mockup нового layout (папки сверху / элементы снизу) в pen
4. Согласовать
5. Реализовать: сначала бэкенд API, потом фронтенд
