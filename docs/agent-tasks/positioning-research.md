# Задача A: Позиционирование и киллер-фича Раскадровки

> Формат: `[Задача A: Позиционирование] — <что делаю сейчас>`
> Это нужно потому что параллельно работают 4 агента и пользователь должен понимать кто есть кто.

## Контекст

**Раскадровка** (`raskadrawka.ru`) — веб-платформа для AI-продакшена. Пользователь создаёт проекты, генерирует изображения и видео через AI-провайдеров (Kie.ai, VEO и др.), организует материалы в группы.

Иерархия: `User → Project → Group → Element (image/video)`

Текущие возможности:
- Мультимодельная генерация (выбор AI-модели, schema-driven параметры)
- Организация в проекты и группы (nested, 2 уровня)
- Drag-and-drop сортировка
- Lightbox с видеоплеером
- Система кредитов ("Заряд") с pricing per model
- Presigned upload напрямую в S3
- WebSocket для real-time обновлений
- Публичный шеринг (в разработке)

Стек: Django + Next.js + Celery + PostgreSQL + S3

## Что нужно сделать

### 1. Анализ рынка
Проведи ресёрч конкурентов. Для каждого определи:
- Основное позиционирование (для кого, зачем)
- Ключевые фичи
- Ценовая модель
- Слабые стороны

Конкуренты для анализа:
- **Leonardo.ai** — AI image generation platform
- **Runway** — AI video generation
- **Pika** — AI video
- **Midjourney** — AI images (Discord-based)
- **Krea.ai** — AI creative tools
- **Viggle** — AI video
- **Kling AI** — video generation

### 2. Определение ниши
На основе анализа ответь:
- Чем Раскадровка отличается от каждого конкурента?
- Что есть у нас, чего нет у других? (мультимодельность, организация в проекты/группы, schema-driven подход)
- Где наша ниша? Кто целевой пользователь?
- Какой messaging? Как объяснить ценность в одном предложении?

### 3. Киллер-фича
Предложи 3-5 потенциальных киллер-фич, для каждой:
- Описание
- Для кого это важно
- Сложность реализации (low/medium/high)
- Уникальность (есть ли у конкурентов)

Учитывай, что планируется:
- Полноценный шеринг с комментариями
- Личный кабинет с аналитикой
- Биллинг (оплата кредитов)

### 4. Рекомендации по roadmap
На основе всего анализа — какие фичи добавлять в первую очередь для максимального отрыва от конкурентов?
