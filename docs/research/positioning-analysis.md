# Позиционирование и киллер-фича Раскадровки

> Дата: 2026-03-27

---

## 1. Анализ конкурентов

### Leonardo.ai
- **Позиционирование:** AI-платформа для генерации изображений. Дизайнеры, гейм-девелоперы, маркетологи.
- **Ключевые фичи:** Собственные модели (Phoenix, Kino XL), Realtime Canvas, fine-tuning, ControlNet, inpainting/outpainting, style reference, upscaling, API.
- **Цены:** Бесплатно 150 токенов/день. Платно от $12/мес (8,500 токенов) до $60/мес (60,000 токенов).
- **Слабости:** Нет организации по проектам/группам — плоская галерея. Нет мультимодельности (только свои модели). Сложный интерфейс. Видеогенерация слабая. Нет коллаборации/шеринга.

### Runway
- **Позиционирование:** Ведущая платформа для AI-видео. Видеомейкеры, кинематографисты, студии.
- **Ключевые фичи:** Gen-3 Alpha / Gen-4, Motion Brush, Camera Control, Lip Sync, Act-One, видеоредактирование, кастомное обучение моделей, API.
- **Цены:** Бесплатно 125 кредитов. Платно от $12/мес до $76/мес (Unlimited).
- **Слабости:** Дорого для активного использования. Нет организации контента. Нет мультимодельности. Медленная генерация. Нет коллаборации.

### Pika
- **Позиционирование:** Простой AI-видеогенератор для широкой аудитории. Блогеры, маркетологи, казуалы.
- **Ключевые фичи:** Pika Effects (Crush, Melt, Explode), Lip Sync, Scene Ingredients, Sound Effects, Reprompt. Низкий порог входа.
- **Цены:** Бесплатно с водяным знаком. Платно от $8/мес до $58/мес (Unlimited).
- **Слабости:** "Фановые" эффекты — не для серьёзного продакшена. Нет генерации изображений. Нет организации контента. Нет мультимодельности. Нет API.

### Midjourney
- **Позиционирование:** Эталон качества AI-изображений. Художники, дизайнеры, архитекторы.
- **Ключевые фичи:** Лучшее качество изображений (V7), уникальная эстетика, Style/Character Reference, персонализация, веб-интерфейс + Discord.
- **Цены:** Без бесплатного плана. От $10/мес до $120/мес.
- **Слабости:** Нет видео. Нет API (бета). Нет организации проектов. Нет мультимодельности. Закрытая экосистема. Нет коллаборации.

### Krea.ai
- **Позиционирование:** Единая креативная платформа с realtime-генерацией. Дизайнеры, маркетологи.
- **Ключевые фичи:** Realtime Generation, Studio Canvas, несколько моделей (частичная мультимодельность), AI Patterns, Logo Illusion, Style Transfer.
- **Цены:** Бесплатно 50 генераций/день. Платно от $24/мес.
- **Слабости:** Realtime = ниже качество. Мультимодельность ограничена. Нет организации контента. Видео базовое. Нет коллаборации.

### Viggle
- **Позиционирование:** Анимация персонажей, вирусный контент. TikTok-блогеры, мемоделы.
- **Ключевые фичи:** Animate, Mix (вставка персонажа в видео), Stylize. Discord + веб.
- **Цены:** Бесплатно с ограничениями. Платно ~$10-25/мес.
- **Слабости:** Узкая специализация. Нестабильное качество. "Мемный" имидж. Нет организации, мультимодельности, API.

### Kling AI
- **Позиционирование:** Мощная и доступная AI-видеогенерация. Прямой конкурент Runway по качеству, дешевле.
- **Ключевые фичи:** Видео до 2 минут, Lip Sync, Motion Brush, Virtual Try-On, Face Swap, генерация изображений, API.
- **Цены:** Бесплатно 66 кредитов/день. Платно от $5.99/мес — агрессивно дёшево.
- **Слабости:** Китайская компания (приватность). Цензура. Нет организации контента. Нет мультимодельности. Нет коллаборации.

## 3. Определение ниши

### Чем Раскадровка отличается от каждого конкурента

- **vs Leonardo/Midjourney/Krea:** Раскадровка — не ещё один генератор. Это **менеджер продакшена** с мультимодельностью. Все конкуренты привязаны к своим моделям.
- **vs Runway/Pika/Kling:** Раскадровка не пытается быть лучшим видеогенератором. Она — **единое окно** для работы с разными провайдерами, с организацией материалов.
- **vs Viggle:** Разные лиги. Viggle — казуальный инструмент для мемов.

### Что есть у нас, чего нет у других

1. **Мультимодельность** — выбор AI-провайдера в одном интерфейсе. Ни один конкурент этого не даёт (Krea — частично).
2. **Организация в проекты и группы** — ни у кого нет. Все хранят контент в хронологической ленте.
3. **Schema-driven подход** — параметры моделей рендерятся динамически. Добавление новой модели не требует изменений frontend.
4. **DnD-сортировка и структурирование** — организация контента как рабочий инструмент, а не галерея.

### Целевой пользователь

**Продюсер AI-контента** — человек, который:
- Работает с несколькими AI-моделями под разные задачи
- Создаёт контент систематически (не разовая генерация, а проекты)
- Нуждается в организации: раскадровки, серии, подборки
- Хочет сравнивать результаты разных моделей
- Работает в команде (шеринг, комментарии)

Примеры: контент-продюсеры, маркетинговые агентства, SMM-специалисты, инди-студии, авторы комиксов / раскадровок.

### Messaging (одно предложение)

> **Раскадровка — единое рабочее пространство для AI-продакшена: генерируй через любую модель, организуй в проекты, делись с командой.**

Альтернативы:
- "Все AI-модели в одном месте. Порядок вместо хаоса."
- "AI-продакшен под контролем: мультимодельная генерация + организация проектов."

## 4. Киллер-фичи — предложения

### 4.1. Мультимодельное A/B сравнение
**Описание:** Один промпт → генерация через 2-3 модели параллельно → side-by-side сравнение результатов. Пользователь выбирает лучший.

- **Для кого:** Продюсеры, которые ищут лучший результат; маркетологи, тестирующие визуальные концепции.
- **Сложность:** Medium (frontend — новый UI сравнения; backend — параллельные задачи уже поддерживаются Celery).
- **Уникальность:** **Ни у кого нет.** Сейчас пользователи переключаются между вкладками разных сервисов вручную.

### 4.2. Шаблоны раскадровок (Storyboard Templates)
**Описание:** Готовые структуры проектов: "Рекламный ролик 30 сек" (5 сцен с описанием), "Instagram серия" (9 карточек), "Комикс 4 кадра". Пользователь выбирает шаблон и заполняет контентом.

- **Для кого:** SMM, маркетологи, блогеры — те, кто не знает с чего начать.
- **Сложность:** Low (JSON-шаблоны + UI выбора, минимум бэкенд-логики).
- **Уникальность:** **Ни у кого нет.** Конкуренты дают пустой холст, не workflow.

### 4.3. Командная работа с комментариями и апрувами
**Описание:** Расшаривание проекта → комментарии к конкретным элементам → статусы (черновик / на ревью / утверждён) → уведомления.

- **Для кого:** Агентства, студии, команды — любой workflow с заказчиком.
- **Сложность:** Medium-High (backend: роли, комментарии, уведомления; frontend: UI ревью).
- **Уникальность:** **Ни у кого нет.** Все конкуренты — single-player. Для командной работы используют внешние инструменты (Figma, Google Drive).

### 4.4. AI-ассистент промптов (Prompt Copilot)
**Описание:** Помощник, который улучшает промпт под конкретную модель: предлагает ключевые слова, оптимизирует формулировку, адаптирует под синтаксис провайдера.

- **Для кого:** Новички и средний уровень — те, кто не знает особенности промптинга для каждой модели.
- **Сложность:** Medium (LLM-вызов для улучшения промпта, UI — inline suggestions).
- **Уникальность:** Частично есть у Leonardo (prompt enhance), но не адаптировано под разные модели. Мультимодельная оптимизация — уникальна.

### 4.5. Автоматические пайплайны (Generate → Upscale → Video)
**Описание:** Цепочка действий: "Сгенерируй изображение моделью X → увеличь через Y → анимируй через Z". Пользователь задаёт пайплайн один раз, дальше — автоматически.

- **Для кого:** Продвинутые пользователи, которые хотят конвейер, а не ручную работу.
- **Сложность:** High (оркестрация задач, UI конструктора пайплайнов, обработка ошибок).
- **Уникальность:** **Ни у кого нет.** Даже ComfyUI (node-based) не предлагает мультипровайдерные пайплайны.

## 5. Рекомендации по roadmap

### Приоритет 1 — Закрепить уникальность (ближайшие 1-2 месяца)

1. **Мультимодельное A/B сравнение** — главный дифференциатор. Простой UX, wow-эффект. Это то, зачем люди придут именно к нам.
2. **Шаблоны раскадровок** — быстро реализуемо, снижает порог входа, усиливает позиционирование "раскадровка", а не "ещё один генератор".
3. **Шеринг с комментариями** (уже в плане) — переводит из single-player в team tool. Критично для B2B.

### Приоритет 2 — Расширение ценности (2-4 месяца)

4. **Prompt Copilot** — повышает конверсию и retention. Снижает фрустрацию новичков.
5. **Биллинг + аналитика** (уже в плане) — монетизация, метрики использования.
6. **Интеграция дополнительных провайдеров** — больше моделей = сильнее мультимодельность. Flux, Stable Diffusion 3, Sora и т.д.

### Приоритет 3 — Платформенность (4-6 месяцев)

7. **Автопайплайны** — высокая сложность, но уникальная фича. Превращает платформу в инструмент для продвинутых.
8. **API для интеграций** — B2B-клиенты, автоматизация.

### Стратегическая рекомендация

**Не конкурировать по качеству генерации** — это война бюджетов, и OpenAI/Google/Runway всегда будут впереди по своим моделям. Вместо этого:

- **Конкурировать по workflow** — организация, сравнение, командная работа
- **Конкурировать по мультимодельности** — быть "агрегатором лучших моделей"
- **Позиция:** не "генератор", а **"рабочий стол для AI-продакшена"**

Аналогия: Figma не рисует лучше Photoshop. Но Figma победила, потому что дала коллаборацию и workflow. Раскадровка может стать "Figma для AI-контента".
